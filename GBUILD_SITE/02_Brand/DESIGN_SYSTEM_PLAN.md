# GBuild Solutions - Design System Plan

## Palette Foundation

Base recommendation:

- `Graphite` for primary text and dark sections
- `Cloud White` for primary light background
- `Stone` for borders and subtle surface separation
- `Slate Mist` for secondary text
- `Deep Teal` as main accent and CTA color
- `Warm Sand` as very restrained supporting note

## Gradient Direction

- use gradients sparingly
- favor soft depth gradients inside dark hero backgrounds or CTA bands
- keep gradients low-noise and readable
- never rely on loud rainbow or neon transitions

## Typography

Recommended families:

- `Geist` or `Satoshi` for primary UI and body
- optional secondary display family only if it adds clear voice without becoming editorial cliche
- `Geist Mono` or `JetBrains Mono` for technical labels or metadata

## Spacing System

- 8px base rhythm
- large section paddings on desktop
- generous spacing between service chapters
- tighter spacing only inside dense proof or data components

## Icon Style

- minimal line or controlled solid icons
- no childish illustration packs
- use icons only when they improve scanning

## Motion Style

- purposeful and smooth
- fast response on hover and press
- stronger transitions for hero and process chapters
- always support reduced motion

## 3D Style

- abstract systems-oriented 3D only
- modular blocks, data cubes, node structures, or architecture-inspired forms
- matte, refined, and low-noise surfaces

## Imagery Style

- product/UI-led when possible
- architectural or systems-adjacent supporting imagery
- real work, interfaces, diagrams, and process visuals over generic stock people

## UI Moodboard Description

A calm but premium digital systems brand: deep structured hero moments, wide confident typography, clean panel composition, subtle gradients, refined motion, and selective technical visualization rather than decorative noise.

## What To Avoid

- Inter by default
- cute illustration libraries
- giant corner radii everywhere
- glossy glassmorphism
- gradients used as decoration without hierarchy benefit
