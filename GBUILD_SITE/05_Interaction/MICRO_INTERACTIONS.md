# GBuild Solutions - Micro Interactions

## Buttons

- hover: subtle lift or tonal shift
- active: small scale-down feedback
- focus: visible accent ring

## Service Cards

- hover should reveal structure, not gimmicks
- slight movement, border emphasis, or image motion is enough

## Navigation

- clear hover state
- smooth active state indicator
- mobile menu should open cleanly and close instantly

## CTA Areas

- strong contrast
- tactile press response
- form success state should feel reassuring and professional

## Forms

- clean focus state
- clear inline validation
- serious, calm success message
